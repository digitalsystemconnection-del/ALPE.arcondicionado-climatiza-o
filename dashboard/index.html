Como o WhatsApp Responde Sozinho?

Para que o WhatsApp responda automaticamente, usamos a API Oficial do WhatsApp Business. Ela permite que um servidor seu receba e envie mensagens de forma programada.

Fluxo de Dados:

  1. Cliente envia mensagem para seu número.
  2. O WhatsApp notifica seu servidor (via Webhook).
  3. Seu servidor consulta a IA (Groq) com a mensagem do cliente.
  4. Seu servidor envia a resposta da IA de volta para o cliente via API.

Exemplo de Código (Servidor com Node.js):

// Este código rodaria em um servidor, não no navegador.

// 1. Rota para receber a mensagem do WhatsApp
app.post('/webhook', async (req, res) => {
  const userMessage = req.body.messages[0].text.body;
  const userPhone = req.body.messages[0].from;

  // 2. Busca resposta da IA (usando a mesma lógica do chat)
  const aiResponse = await buscarRespostaIA(userMessage);

  // 3. Envia a resposta de volta para o usuário via API
  await enviarMensagemWhatsApp(userPhone, aiResponse);

  res.sendStatus(200); // Informa ao WhatsApp que a mensagem foi recebida
});
